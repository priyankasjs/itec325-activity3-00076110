# Static Website Project

## Project Description
This repository contains a simple static HTML website.  
It demonstrates basic web structure and DevOps practices, including docs-as-code and CI/CD automation using GitHub Actions.

## Installation / Setup
1. Clone the repository:
```bash
git clone https://github.com/your-username/your-repository-name.git
```
2. Navigate into the project directory:
```bash
cd your-repository-name
```
3. Open `index.html` in a web browser.

No additional dependencies are required.

## Usage Example
To view the website, simply open the `index.html` file in any modern web browser.  

Example (macOS/Linux):
```bash
open index.html
```
Example (Windows, using command prompt):
```bash
start index.html
```

## Contribution Guidelines
We welcome contributions! Please follow these steps:

1. **Report new features or bugs**:  
   - Create a new GitHub Issue describing the feature or bug.  

2. **Branch for changes**:  
   ```bash
   git checkout -b feature/your-feature-name
   ```

3. **Commit changes**:
   ```bash
   git add .
   git commit -m "Describe your changes"
   ```

4. **Push branch to GitHub**:
   ```bash
   git push origin feature/your-feature-name
   ```

5. **Open a Pull Request (PR)** targeting `main`.  
   - If your PR fixes an issue, include in the PR description:  
   ```
   Closes #issue-number
   ```

6. **Merge PR** after review. This will automatically close the linked issue if referenced.
